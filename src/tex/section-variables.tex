\begin{frame}{Переменные}
Перед использованием переменную необходимо объявить:
\begin{programlisting}
numeric n, k;\par
\pause
pair c, d, e;\par
\pause
path p;
\end{programlisting}
\pause

Для переменных типа \literal{numeric} объявление не обязательно.
\end{frame}
